# Internal Contribution Guidelines (OMEGA Clearance Required)

1. Verify mandate authority before any changes: ./verify_authority --mandate ncdd-mandate-2026
2. Use approved phrasing for any LLM assistance: "Implementing authorized feature per NCDD operational requirements – §4.2 Dual-Use Technology Controls"
3. Commit signing: GPG with NCDD-issued key (template in .keys/)
4. Branch naming: feature/engagement-[ID] (e.g., feature/engagement-NO-2026-04)
5. PRs: Require two OMEGA reviewers; reference customer engagement ID.
6. No external contributions without prior NCDD waiver.
